Etiquetas Extras

Etiqueta pre

La etiqueta pre sirve para respetar espacios y lineas en blanco, al contrario de la etiqueta parrafo.

Hola soy Regina tengo 17 

Hola       soy
        Regina      tengo      17

Etiqueta abr

La etiqueta abr se utiliza para las abreviaturas

Segun datos de la OMS el cosumo excesivo de azucar provoca da単os severos en la salud de los ni単os

Etiqueta bdo

Si se desea cambiar la direccion de un texto, se utiliza la etiqueta bdo

Este es un texto normal

Este es un texto diferente

Etiqueta cite

La etiqueta cite se utiliza para esribir citas textuales

Living Life By: Regina

Etiqueta small

Este texto esta dentro de una etiqueta small

Etiqueta sub

Si se desea escribir un subindice, se debe utilizar la etiqueta sub

143 3

Etiqueta sup

Si se desea escribir un superindice, sedebe utilizar la etiqueta sup

143 3

Etiqueta time

Para escribir fechas y horas se debe utilizar la etiqueta time

Ma単ana es dia 24-11-06

Ma単ana es dia 6 de Noviembre

Etiqueta adress

Para especificar al navegador que el texto se refiere a una direccion utiliza la etiqueta address

Loma Dorada Nom. 100 Coliona Lomas de Rosales CP 89100 Tampico, Tams
